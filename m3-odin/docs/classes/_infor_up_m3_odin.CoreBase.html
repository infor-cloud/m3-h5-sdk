Class CoreBase

Optional base class with logging functions.

import { CoreBase } from '@infor-up/m3-odin';

Extending classes must call the super function in the constructor and provide the name of the type. The type name is used in the console log output. The CoreBase class will automatically create a log entry on the debug log level when the constructor is called.

Example

export class MyCustomClass extends CoreBase {
   constructor() {
      super('MyCustomClass');
   }

   onClick(): void {
      this.logInfo('onClick: Called');
   }
}
