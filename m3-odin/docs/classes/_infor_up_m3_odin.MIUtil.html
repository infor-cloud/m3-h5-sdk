Static createUpdateRecord

  • createUpdateRecord(originalValues, newRecord, fieldNames, mandatoryFields): MIRecord

  • Creates a MIRecord that contains only the updated fields and the mandatory keys. Should be used when making an update transaction so that unchanged values are not set by the update transaction. Note that the values must have the correct type. Dates must be date objects and numeric values must be numbers.

    Parameters

    • originalValues: MIRecord

      A MIRecord with the original values, for example the values from a Get or List transaction.

    • newRecord: any

      A MIRecord or object with parameters for all values using the correct data type for the transaction.

    • fieldNames: string[]

      An array with field names that should be updated (if the value has changed). This list does not need to contain the mandatory fields.

    • mandatoryFields: string[]

      An array with mandatoryFields. The mandatory fields should always be added to the new record.

    Returns MIRecord

    A MIRecord that should be used in an update transaction.
