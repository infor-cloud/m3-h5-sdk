Optional refresh

refresh: boolean
  • Defined in core/types.ts:164

Gets or sets an optional value that indicates if the ION API token should be refreshed or not.

The default value is false. This property should only be set to true if the current token is invalid. An ION API call will return HTTP 401 when the token is invalid.
