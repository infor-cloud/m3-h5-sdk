Interface IEnvironmentContext

Defines environment context values. The values are also available on the user context if the application is running in a H5 tab. This context is only useful when there is an application without an existing H5 session but there is still the need to get the ION Api URL. Calling this method might result in a login if the data isn't already available.

import { IEnvironmentContext } from '@infor-up/m3-odin';
