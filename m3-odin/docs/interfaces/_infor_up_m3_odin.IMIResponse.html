Interface IMIResponse

Represents the response from an M3 MI transaction.

import { IMIResponse } from '@infor-up/m3-odin';

The response contains a single MIRecord or a list of MIRecords that can be accesses through the item and items properties. Note that the type of the item and items properties are any and any[] to simplify field value access.

If the request has [[IMIRequest.typedOutput]] set to true all values in the response will be typed according to the [[MIDataType]], (String, Numeric and Date). The default is that all values are strings.

Example

this.miService.execute(request).subscribe((response: IMIResponse) => {
   const item = response.item
   if (item) {
      this.updateName(item.NAME);
   }
}
