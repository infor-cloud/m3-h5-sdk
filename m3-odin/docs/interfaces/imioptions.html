Interface IMIOptions

Represents options for executing an MI request. See IMIRequest.

import { IMIOptions } from '@infor-up/m3-odin';

Example

const request = {
   program: 'CRS610MI',
   transaction: 'LstByNumber',
   excludeEmptyValues: true,
   maxReturnedRecords: 10
} as IMIRequest;

Optional includeMetadata

Optional maxReturnedRecords

maxReturnedRecords: number
  • Defined in projects/infor-up/m3-odin/mi/types.ts:62

Gets or sets the maximum number of records to return. Always check maxReturnedRecordsFieldSpecified before using this value since 0 means that all rows should be returned.

Optional tag

tag: any
  • Defined in projects/infor-up/m3-odin/mi/types.ts:68

Gets or sets an arbitrary object value that can be used to store custom information about this request. The value will be returned in the tag property of the MIReponse object.

Optional typedOutput

typedOutput: boolean
  • Defined in projects/infor-up/m3-odin/mi/types.ts:82

Gets or sets a value that indicates if output should be converted to numbers and dates according to the metadata definition for the MI transaction. This implicitly turns on includeMetadata in the options to load the metadata information. The default value is false.
